Web III: Assignment 02 GitHub

This assignment uses Angular Material. It is an implementation of Google Material Design and uses some FlexBox compenents. The assignment also uses the EJS engine for rendering pages and gives us a PHP-esque system for outputing content and logic to the HTML document. The login system uses the passport module to handle sessions and authentication.
